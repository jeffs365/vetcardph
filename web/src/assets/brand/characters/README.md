# VetCard Character Assets

These are the reusable VetCard character models for product UI, onboarding, empty states, and light engagement moments.

## Files

- `vetcard-dog.png` — transparent dog cutout
- `vetcard-cat.png` — transparent cat cutout
- `vetcard-parrot.png` — transparent parrot cutout
- `vetcard-character-lineup.png` — transparent lineup of all three characters
- `vetcard-character-lineup-chroma.png` — original generated chroma-key source
- `index.ts` — import/export helper for app code

## Style Notes

- Cute polished mobile-app illustration
- Clean ink outlines
- Warm orange/white pet palette
- Teal collars and VetCard-adjacent tags/details
- Friendly veterinary-care tone
- Transparent PNGs for flexible placement

Use these as the visual reference when generating future poses, expressions, or scene variants.

## Original Prompt

```text
Create a polished reusable character lineup matching the existing VetCard login illustration style: one friendly orange-and-white dog with teal collar and small medical/paw tag, one friendly orange tabby cat with white chest and warm expression, and one cheerful yellow-green parrot with teal wing feathers. These are the app mascot models and must feel consistent with the current VetCard cute login hero characters.

Three separate full-body characters standing side by side with generous space between them, no overlap, each fully visible from head to feet/tail, centered in a square image.

Cute polished mobile-app illustration, clean ink outlines, soft warm highlights, subtle depth inside the character only, friendly veterinary/pet-care feeling, high quality, not childish scribble.

Perfectly flat solid #ff00ff chroma-key background for background removal.
```
